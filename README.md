## Our Plugin Suite

### CraftersStaff — Staff Management System

The complete solution for professional server moderation teams.

**What it does:**
CraftersStaff transforms how your moderation team operates. From silent monitoring to active intervention, it provides every tool your staff needs in one cohesive system.

**Key Capabilities:**
- **Vanish System** — Complete invisibility with multiple visibility tiers for undercover monitoring
- **Freeze Mechanism** — Instantly immobilize players under investigation
- **Staff Communication** — Private chat channels separate from public chat
- **Player Notes** — Attach persistent notes to players with status tracking
- **Activity Logging** — Complete audit trails of all staff actions
- **Report Management** — Handle player reports with organized case tracking

**Perfect for:** Servers with active moderation teams who need coordinated tools

---

### CraftersBan — Advanced Punishment System

Professional-grade discipline management for Minecraft networks.

**What it does:**
CraftersBan provides comprehensive punishment tools that scale from small servers to massive networks. Every action is tracked, synchronized, and reversible.

**Key Capabilities:**
- **Flexible Bans** — Temporary, permanent, and IP-based banning
- **Mute System** — Chat moderation with configurable durations
- **Warning Accumulation** — Progressive discipline with automatic escalation
- **Network Synchronization** — Punishments sync across all backend servers
- **Detailed History** — Complete lookup of player infraction records
- **Silent Mode** — Execute actions without public broadcast

**Perfect for:** Servers and networks that need consistent, fair moderation across multiple game modes

---

### CraftersLogin — Authentication System

Enterprise-grade player authentication for secure Minecraft networks.

**What it does:**
CraftersLogin delivers professional authentication for servers requiring complete access control. From cracked servers to premium networks, it handles every scenario.

**Key Capabilities:**
- **Secure Authentication** — Industry-standard password hashing (BCrypt, Argon2)
- **Premium Integration** — Automatic Mojang/Microsoft account verification
- **Proxy Support** — Full Velocity, BungeeCord, and Waterfall compatibility
- **Database Flexibility** — SQLite, MySQL, or MongoDB options
- **Brute Force Protection** — Rate limiting and IP-based security
- **Session Management** — Secure tokens with hijacking prevention
- **Customizable Experience** — Titles, action bars, sounds, and messages

**Perfect for:** Networks requiring secure player authentication across multiple servers
